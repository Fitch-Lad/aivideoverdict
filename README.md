# AI Video Verdict

Independent, hands-on reviews of the best AI video tools of 2026.

**Live site:** https://aivideoverdict.com

A static site — plain HTML/CSS, no build step. Deploys straight to **GitHub Pages** or
**Cloudflare Pages** from the repository root (`index.html` is the entry point).

## Pages
- `index.html` — the review & comparison hub
- `links.html` — link-in-bio page for social traffic

## Deploy
**GitHub Pages:** Settings → Pages → Deploy from branch → `main` / root.
**Cloudflare Pages:** connect this repo, framework preset = none, output directory = `/`.
